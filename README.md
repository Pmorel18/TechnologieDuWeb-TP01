# Questions
## 1. Expliquez ce qu’est le Layout Shift.
Le Layout Shift, ou décalage de mise en page, est un phénomène qui se produit lorsque les éléments d'une page web changent de position de manière inattendue pendant le chargement ou l'interaction de l'utilisateur. Cela peut être causé par des images ou des vidéos qui se chargent après le texte, des polices qui s'affichent tardivement, ou des éléments dynamiques qui modifient la structure de la page. Le Layout Shift peut entraîner une mauvaise expérience utilisateur, car il peut provoquer des clics accidentels sur des éléments déplacés ou rendre la navigation difficile. C'est pourquoi il est important de minimiser les Layout Shifts pour améliorer la stabilité visuelle d'une page web.

## 2. Expliquez à quoi sert aria-label.
L'attribut `aria-label` est utilisé pour fournir une description textuelle aux éléments interactifs ou non textuels d'une page web, pour les rendre accessibles aux utilisateurs avec de l'assistance comme des lecteurs d'écran.
